# Facebook Login Clone 
A pixel-perfect clone of the Facebook login page built with pure HTML and CSS.

## Features
- Facebook-accurate color scheme and layout
- Responsive input fields with focus states
- Hover animations on buttons
- Clean, minimal code structure
